			✅ Python Data Types – Mutability & Address Behavior
			| Data Type | Mutable / Immutable | Value Change Possible? | Address Change on Modify   | Example                   |
			| --------- | ------------------- | ---------------------- | -------------------------- | ------------------------- |
			| `int`     | ❌ Immutable         | ❌ No                   | ✅ Yes (new object created) | `a = 10 → a = 20`         |
			| `float`   | ❌ Immutable         | ❌ No                   | ✅ Yes                      | `a = 1.5 → a = 2.5`       |
			| `string`  | ❌ Immutable         | ❌ No                   | ✅ Yes                      | `"hi" → "hello"`          |
			| `list`    | ✅ Mutable           | ✅ Yes                  | ❌ No (same object)         | `a = [1,2] → a.append(3)` |
			| `tuple`   | ❌ Immutable         | ❌ No                   | ✅ Yes                      | `(1,2) → (1,2,3)`         |
